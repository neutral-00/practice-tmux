# Switch Session

ctrl-b s


use j and k to navigate in the list and switch by pressing enter

Another way is to press ctrl-b and
1. ) to go to then next session
1. ( to go to then previous session


Yet another better way to navigate is to use ctrl-b w

This will show you sessions > windows > panes

So you can navigate to a specific session > window > pane.

That is precision.
